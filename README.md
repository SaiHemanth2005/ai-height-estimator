# AI Height Estimation System

This project estimates a person's height using a single camera and AI-based pose detection.

## 🚀 Features
- Real-time height detection
- Auto calibration
- Distance validation
- Clean UI overlay

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run:
python main.py
